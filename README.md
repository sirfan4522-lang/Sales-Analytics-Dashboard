# 📊 Sales Analytics Dashboard

## Project Overview
This Power BI Sales Analytics Dashboard provides insights into sales performance, customer behavior, product categories, and monthly trends.

## Key Features
- Total Sales Analysis
- Customer Analysis
- Transaction Analysis
- Average Order Value (AOV)
- Product Category Performance
- Sales by Gender
- Sales by Age Group
- Monthly Sales Trends
- Interactive Filters and Slicers

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Business Insights

- Groceries generated the highest sales.
- The 31–40 age group contributed the most revenue.
- Repeat customers accounted for the majority of sales.
- Monthly sales peaked during May and July.
