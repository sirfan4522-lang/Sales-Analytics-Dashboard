# 📈 Sales Analytics Dashboard

## Project Objective

To analyze sales performance, customer behavior, product category trends, and purchasing patterns using Power BI. The dashboard helps businesses track key sales KPIs and identify growth opportunities.

---

## Project Overview

This Power BI Sales Analytics Dashboard provides insights into revenue performance, customer segmentation, transaction analysis, product category sales, basket size trends, demographic purchasing behavior, and monthly sales performance. Interactive filters allow users to drill down into customer and product-level insights.

---

## Key Features

- Total Sales Analysis
- Customer Performance Tracking
- Transaction Analysis
- Average Order Value (AOV) Monitoring
- Average Basket Size Analysis
- Product Category Performance
- Gender-wise Sales Distribution
- Age Group Sales Analysis
- Customer Type Analysis
- Monthly Sales Trend Analysis
- Interactive Filters and Slicers

## Business Insights

- Total sales reached **574K** across **1,000 transactions**.
- The business served **269 customers**, generating an average order value of **510**.
- Repeat customers accounted for nearly **69% of total customers**, indicating strong customer retention.
- Customers aged **31–40 years** contributed the highest sales volume among all age groups.
- Male and female customers generated relatively balanced sales contributions.
- Groceries emerged as the highest-performing product category in terms of sales.
- Electronics achieved the highest average basket size, indicating higher-value purchases.
- Sales peaked during **May and July**, suggesting seasonal demand patterns.
- Monthly sales fluctuations highlight opportunities for targeted promotional campaigns during low-performing periods.
